# EventSnap — MVP

**Дедлайн: 7 апреля 2026**

## Цель MVP

Рабочий веб-сервис где можно создать событие, пригласить людей через QR,
загрузить фотографии и посмотреть общую галерею.

## Функциональные требования

### Пользователи
- [ ] Регистрация по email + password
- [ ] Вход по email + password
- [ ] Профиль пользователя (имя, аватар)

### События
- [ ] Создать событие (название, описание, дата, место)
- [ ] Редактировать событие
- [ ] Удалить событие (soft delete)
- [ ] Список своих событий
- [ ] Обложка события

### QR и участники
- [ ] Генерация QR-кода для события
- [ ] Вход в событие по QR (автоматическое добавление участником)
- [ ] Список участников события

### Галерея и фото
- [ ] Загрузка фото в галерею события
- [ ] Просмотр галереи (сетка одобренных фото)
- [ ] Удаление своего фото
- [ ] Настройка модерации галереи (вкл/выкл)

### Модерация
- [ ] Организатор видит очередь фото на проверку
- [ ] Одобрить / отклонить фото с комментарием
- [ ] Если модерация выключена — фото публикуется сразу

## Нефункциональные требования

- Работает в браузере на мобильном устройстве (адаптивная вёрстка)
- Фото хранятся в S3-совместимом хранилище
- Thumbnail генерируется автоматически для быстрой загрузки галереи

## Вне скоупа MVP

- OAuth через Apple
- Скачивание архива галереи
- Уведомления
- Приватные ссылки с паролем
- Монетизация
- Мобильное приложение
