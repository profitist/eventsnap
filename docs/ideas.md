# EventSnap — Идеи и видение продукта

## Суть продукта

EventSnap — веб-приложение для хранения и просмотра фотографий с личных мероприятий.
Корпоративы, домашние посиделки, дни рождения, встречи друзей — всё в одном месте,
аккуратно разложено по событиям.

## Проблема которую решаем

После любого мероприятия фотографии разбросаны по телефонам участников.
Кто-то скидывает в общий чат, кто-то нет — в итоге общая картина никогда не собирается.
EventSnap даёт одно место куда все скидывают фото, и откуда все их смотрят.

## Ключевая механика

- Организатор создаёт событие и получает QR-код
- Гости сканируют QR — попадают в галерею события
- Каждый загружает свои фото прямо с телефона через браузер
- Все видят общую галерею со всеми фотографиями

## Чем отличаемся

- Не требует установки приложения — работает в браузере
- QR-код как точка входа — просто и понятно для любой аудитории
- Фокус на мероприятиях, не на личном хранилище
- Модерация фото на усмотрение организатора

## Идеи для будущего (не MVP)

- Скачивание всей галереи одним архивом
- Автоматическая группировка фото по времени съёмки
- Уведомления когда добавили новые фото
- Приватные ссылки с паролем
- Монетизация: платные тарифы с увеличенным хранилищем или кастомным брендингом
